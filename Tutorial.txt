Flask Tutorial:
blueprints: in all unique folders(main,posts,users), a __init__.py file is required

flask tutorial on forms and validation
- pip install flask-wtf(in cmd)
- pip install flask-flash

database with Flask-SQLAlchemy
- we will be using a SQLite database for development
- pip install flask-sqlalchemy
- pip install pylint-flask (if pylint is not installed on your computer, otherwise can ignore)

user authentication/hashing algorithms (hashing passwords and sending to DB)
- pip install flask-bcrypt
- pip install flask-login

to reformat images 
- pip install pillow

to send email in flask
- pip install flask-mail
